🎵 MusicStore - Aplicación Web de Tienda Musical
🧾 Descripción

MusicStore es una aplicación web desarrollada con React que simula una tienda virtual de productos musicales. Permite visualizar, gestionar y simular la compra de artículos, aplicando buenas prácticas de desarrollo frontend como componentización, manejo de estado y diseño responsive.

✨ Características principales
🎸 Visualización de productos en tarjetas interactivas
✏️ Edición y eliminación de productos
🔎 Barra de búsqueda dinámica
🛒 Sistema de carrito de compras
💳 Formulario de datos del cliente y simulación de pago
📱 Diseño responsive adaptable a distintos dispositivos
🖥️ Interfaz gráfica

La aplicación está organizada en secciones clave:

Header: Logo, buscador y acceso al carrito
Contenido principal: Listado de productos con imagen, precio y acciones
Panel lateral: Gestión del carrito y formulario de compra
Footer: Información del proyecto y del autor
🏗️ Arquitectura del proyecto

El proyecto sigue una estructura modular basada en componentes reutilizables:

src/
 ├── features/layout/components/
 │   ├── Header.jsx
 │   ├── Content.jsx
 │   ├── Footer.jsx
 │   └── Props.jsx
 ├── shared/styles/
 ├── App.jsx
 └── main.jsx

🧠 Conceptos aplicados
Componentización en React
Uso de props para comunicación entre componentes
Navegación con React Router DOM
Separación de responsabilidades
Diseño escalable y mantenible

🚀 Instalación y ejecución
Clonar el repositorio:
git clone https://github.com/tu-usuario/musicstore.git
Entrar al proyecto:
cd musicstore
Instalar dependencias:
npm install
Ejecutar el proyecto:
npm run dev
